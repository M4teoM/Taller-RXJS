# Taller RxJS

## 1) Instalacion

```bash
npm install
```

## 2) Ejecutar por ejercicio

```bash
npm run ej1
npm run ej2
npm run ej3
npm run ej4
npm run ej5
npm run ej6
```

## 3) Verificacion de tipos

```bash
npm run check
```

## Estructura

- `src/index.ts`: punto de entrada general.
- `src/ejercicios/`: archivos base de cada tema.
- `src/recursos/`: datos y utilidades compartidas.
